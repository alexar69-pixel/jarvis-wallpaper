=========================================
JARVIS STARK HUD ASSISTANT - WALLPAPER ENGINE
=========================================

INSTRUCCIONES DE USO:
1. Asegúrate de tener micrófono conectado.
2. Haz doble clic en el archivo `Instalar_Jarvis.bat`. Esto configurará todo automáticamente y abrirá la ventana de Ajustes.
3. En la ventana de Ajustes, ingresa tu API Key de Gemini u OpenAI.
4. Una vez configurado, cada vez que enciendas tu PC, haz doble clic en `Iniciar_Jarvis.bat` para encender el "Cerebro" del asistente.
5. Aplica este fondo en Wallpaper Engine, activa la opción "Interacción de ratón" para este fondo, ¡y di "Jarvis"!

NOTA: Al ser un script compilado que lee sensores de tu sistema y escucha tu micrófono, es posible que Windows Defender o tu Antivirus te pregunten si permites la ejecución o el acceso a la red. Dale a "Permitir".
